# v0.3.0 Boundary Summary

`v0.3.0` is the release that closes the currently defined roadmap through
Milestone K.

## Included

- deterministic bundle ingest, retrieval, runtime, and artifact flow
- pluggable fixture registry
- provider plugin registry
- local `gemma4` answer plugin
- optional hosted provider comparison flow
- release automation and release-candidate gating
- explicit Chronicle write-back boundary closure

## Still Optional

- `gemma4` runtime use
- hosted provider runtime use
- provider plugin tests
- optional plugin compatibility matrix checks during release

## Explicitly Out of Scope

- Chronicle write-back from this repository
- treating downstream artifacts as authoritative Chronicle facts
- requiring any hosted provider for the baseline release path
