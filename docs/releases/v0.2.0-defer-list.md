# v0.2.0 Deferred Items

This file records items explicitly deferred from the first supported baseline.

## Deferred by Design

- hosted vector-provider integrations remain out of scope for `v0.2.0`
- provider-specific tests and credentials remain out of scope for `v0.2.0`
- Chronicle-side write-back or trial-record persistence from this repository
  remains out of scope for `v0.2.0`
- broader fixture expansion beyond the committed minimal and representative
  sanitized bundles remains out of scope unless Chronicle-side contract shape
  changes
- release automation beyond the current smoke-based CI path remains out of
  scope for `v0.2.0`

## Not Deferred

- ingest contract hardening for the supported bundle shape
- graph-only and hybrid retrieval with deterministic local fixtures
- runtime answers, saved evaluation artifacts, and markdown reports
- reproducible clean-checkout validation for the supported local-only path
